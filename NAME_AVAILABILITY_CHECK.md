## üîç How to Verify Name Availability

Before finalizing any name, check:

1. **Trademark Search**
   - USPTO.gov (US Patent and Trademark Office)
   - EUIPO.eu (European Union)
   - WIPO.int (International)

2. **Domain Availability**
   - Check .com, .io, .app domains
   - Use: namecheap.com, godaddy.com

3. **App Store Search**
   - Apple App Store
   - Google Play Store
   - Search exact name and variations

4. **Social Media Handles**
   - Twitter/X: @name
   - Instagram: @name
   - Facebook: /name
   - LinkedIn: company/name

5. **Web Search**
   - Google search: "name" + "app" + "therapy"
   - Check for existing websites/services
